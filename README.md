# Airflow Weather Data Pipeline

This project implements an automated data pipeline using Apache Airflow, Docker, PostgreSQL, and Python.

## Features
- Extracts weather data from Open-Meteo API
- Schedules daily execution using Airflow DAG
- Stores data in PostgreSQL
- Visualizes trends using Python

## Tech Stack
- Apache Airflow
- Docker
- PostgreSQL
- Python (Pandas, Matplotlib)

## How to Run
1. Run docker compose up -d
2. Access Airflow at http://localhost:8080
3. Trigger DAG
